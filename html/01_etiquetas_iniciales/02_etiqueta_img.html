Etiquetas de imagen: "IMG"


Cuando queremos mostrar una imagen en un documento HTML necesitamos utilizar la etiqueta:

<img>

El código 

<img>
no es necesario cerrarla como las etiquetas de Headings, por ejemplo 
<h1>

Esta etiqueta necesita utilizar atributos. En este caso el principal es 'src' para indicar la fuente o 'source' de la imagen a mostrar.

Una forma clásica donde aparece esta etiqueta con su atributo es la siguiente: 

<img scr='imagen.png'>
Aunque este no es el único atributo de esta etiqueta.
