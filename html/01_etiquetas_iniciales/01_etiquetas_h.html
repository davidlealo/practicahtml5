HTML Headings


Tenemos seis tipos de Headings o textos destacados, los cuales podemos usar, por ejemplo, como títulos.

Estos son: h1, h2, h3, h4, h5 y por último h6.

Números de Headings inferiores a 1 o superiores a 6 no dan resultado.


Ejemplos exitosos

Este es un H1

Este es un H2

Este es un H3

Este es un H4

Este es un H5
Este es un H6

Ejemplos de Headings con números incorrectos

Este es un H0
Este es un H-1
Este es un H7
Este es un H516
